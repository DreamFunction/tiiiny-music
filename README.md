## 介绍

tiiiny music 是一个极简音乐 DAW，准确说应该叫 tiiiiiiiiiiiiiiiiiiiiiiiiiiiiii...（自己想象有多少个 i）ny music，主打极简、搞笑（但也正经），中文名叫极————————————————————（自己想象有多少个“——”）……简 音乐。

**为啥叫这么个名？**
> 因为我这是 **真·极简**，为了让大家印象深刻，不至于已经被“极简”说疲劳，所以叫这个名字。正经名字总不能真打这么多字，所以叫 tiiiny music。作者本人很讨厌一堆乱七八糟的功能，要写个歌还要：一顿选择，一顿习惯，想找个精简的，要么还是嫌太重，要么根本无法作曲（至少我还没见过类似项目）。所以干脆自己开发了一个，由于深受功能过剩之害，因此下定决心要极简。这个项目可能到最后也不会有太庞大的功能。有的轻型需求用这种 **真·微型 DAW** 就不错。

> 请记住：**再念这个名字的时候，请自动延音 30 秒**（哈哈，开玩笑的）

作者没重度使用，但是亲自浅尝：可在作者的香橙派 Zero3（类似树莓派 Zero2W，但是性能稍强，频率 1.5GHz，内存 1GB）上流畅运行（所以估计在笔记本上也不会有性能问题）。

**核心理念**：就写歌，别的麻烦事啥也没有。

> 本版本还不完善，但是功能已经够用（”先跑起来再说“），可以写个《小星星》什么的。

## 安装教程

暂时没有，靠自己部署吧。要有 Python 环境，第三方库：`notemidi`（作者自己的库哦），`mido`，安装 `fluidsynth`，然后启动才能出声。

## 使用说明

呃……界面都简单成这样了，点进去应该就能知道怎么用吧？不过还是讲一下吧：

打开和保存，目前只支持同级目录下保存为或打开 `data.json`。3 个下拉菜单：”c“那列是音名，”4“是哪一组（1-8），c4 是中央 C。最后那个是时值，比如 4 指四分音符。菜单编辑完，按”添加“放进旋律列表，选中并按”删除“可以删掉一个项目。按“播放”就能播放。
